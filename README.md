# 🎓 EduPanel

EduPanel is a **Blazor Server + MudBlazor UI playground** built for learning modern .NET UI development.

This project focuses purely on **UI/UX design and component mastery** using MudBlazor, without a database or backend complexity.

---

## 🚀 Tech Stack

* .NET 9 (Blazor Web App / Server Interactive)
* MudBlazor (UI Components)
* Heron.MudCalendar (Calendar Component)
* bUnit (Unit Testing for Blazor)
* C#
* Razor Components

---

## 🎯 Purpose

This project is designed to:

* Learn Blazor component architecture
* Practice MudBlazor UI components
* Build modern dashboard layouts
* Experiment with authentication UI (no database)
* Improve frontend design skills in .NET

---

## 🔐 Features (Planned)

* Login Page (hardcoded authentication)
* Admin Dashboard Layout
* Sidebar Navigation
* **Employee Portal**:
  * Profile Overview
  * Personal Data Update (Multi-tab structure)
  * Attendance Management (Calendar, Leave, DTR, Overtime)
* Student Management UI (mock data)
* Teacher Management UI (mock data)
* Settings Page
* Responsive Design
* Dark/Light Mode

---

## 🧪 Current Status

🚧 In Development — Building core layouts, employee portal features, and focusing on a clean **Component-Based Architecture**.
